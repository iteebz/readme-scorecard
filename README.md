# README Scorecard

Paste a public GitHub repo URL. Get an instant README grade — shareable, no install.

**[Try it → iteebz.github.io/readme-scorecard](https://iteebz.github.io/readme-scorecard/)**

## What it checks

- README exists and has sufficient length
- Has a title, description, installation and usage sections
- Includes code examples and license
- Has badges and contributing/resources section
- Overall structure (number of sections)

Scores 0–100, grades A through F. Every check shows pass / warn / fail with a reason.

## Usage

No install. Open the page, paste any public GitHub repo URL, hit Grade it.

The result page has a permalink — `?repo=owner/name` — so you can share your grade directly.

## Share

The permalink IS the artifact. Share it on X, paste it in a PR, drop it in an issue. Anyone who clicks sees the grade card and can score their own repo.
